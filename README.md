# Saksham Fashion Zone - Inauguration Website

A premium saree & blouse showroom website built with Next.js featuring Indian ethnic luxury design.

## Features

- **Inauguration Landing Page** with countdown timer
- **Product Listing** with advanced filters
- **Product Detail Pages** with combo offers
- **Special Offers Page** with flash deals
- **Admin Panel** for inventory management
- **Responsive Design** for all devices
- **Indian Ethnic Theme** with luxury aesthetics

## Tech Stack

- Next.js 14
- React 18
- Tailwind CSS (via custom CSS)
- Lucide React Icons
- Framer Motion (for animations)

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## Pages

- `/` - Homepage with inauguration theme
- `/products` - Product listing with filters
- `/product-detail` - Individual product pages
- `/offers` - Special inauguration offers
- `/about` - Brand story and values
- `/contact` - Store location and contact
- `/admin` - Admin panel for management
